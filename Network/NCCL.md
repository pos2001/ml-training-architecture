
### NCCL
```
NCCL = GPU 간 통신을 위한 NVIDIA 라이브러리

용도:
- 멀티 GPU 학습 시 데이터 동기화
- GPU 간 텐서 교환
- 분산 딥러닝 통신

통신 패턴:
├── All-Reduce: 모든 GPU에서 결과 집계
├── Broadcast: 한 GPU에서 모든 GPU로 전송
├── Reduce: 여러 GPU 결과를 하나로 합침
└── All-Gather: 모든 GPU 데이터 수집

```

```
┌─────────────────────────────────────────┐

└─────────────────────────────────────────┘
↓
┌─────────────────────────────────────────┐
│ EFA (Elastic Fabric Adapter)            │

│ - 낮은 지연시간                          │
└─────────────────────────────────────────┘

```
