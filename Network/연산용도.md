### 일반적인 학습 (GPT, BERT 등)
```
✅ AllReduce만 테스트해도 OK!

이유:
├─ 대부분의 딥러닝은 AllReduce가 핵심
├─ 그래디언트 동기화 = AllReduce
├─ 가장 많이 사용되는 연산
└─ 병목이 여기서 발생
```






### 특수한 경우 (MoE 모델 등)
```
⚠️ AllToAll도 테스트 필요!

이유:
├─ MoE (Mixture of Experts) 모델
├─ 전문가(Expert) 라우팅 = AllToAll
├─ 레이턴시에 매우 민감
└─ AllReduce와 다른 특성
```
