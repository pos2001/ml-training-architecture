

## MPI 통신 주체? 프로세스

### MPI 구조
```
물리적 구조:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

노드 1:
  CPU:
    ├─ 코어 0
    ├─ 코어 1
    ├─ 코어 2
    └─ 코어 3

프로세스 배치:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

옵션 1: 프로세스당 1개 코어
  프로세스 0 → 코어 0
  프로세스 1 → 코어 1
  프로세스 2 → 코어 2
  프로세스 3 → 코어 3

옵션 2: 프로세스당 2개 코어 (멀티스레드)
  프로세스 0 → 코어 0, 1
  프로세스 1 → 코어 2, 3

옵션 3: 프로세스 1개가 모든 코어 사용
  프로세스 0 → 코어 0, 1, 2, 3

```
