

### Forward Pass (순전파)
### "입력 → 출력" 방향으로 계산, 입력 데이터를 모델에 넣어서 예측값을 계산하는 과정

```
입력(X) → Layer 1 → Layer 2 → Layer 3 → 출력(Ŷ)
   ↓         ↓         ↓         ↓         ↓
  이미지    특징추출   특징조합   분류     예측결과

```


### Backward Pass (역전파)
### "출력 → 입력" 방향으로 gradient 계산
```
예측 오차를 각 파라미터(W, b)에 대해 미분하여
어떻게 수정해야 할지 계산하는 과정

Loss ← 출력(Ŷ) ← Layer 3 ← Layer 2 ← Layer 1 ← 입력(X)
  ↓         ↓         ↓         ↓         ↓
∂L/∂Ŷ   ∂L/∂W₃    ∂L/∂W₂    ∂L/∂W₁   (gradient)

```




## 시각적 비교
### Forward Pass 상세
```
┌─────────────────────────────────────────────────────────┐
│                    Forward Pass                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  입력 이미지 (고양이 사진)                               │
│  ┌─────────┐                                            │
│  │ 🐱      │  x = [0.2, 0.8, 0.1, ...]                 │
│  └─────────┘                                            │
│       ↓                                                 │
│  Layer 1: 특징 추출                                     │
│  ┌─────────────────────────────┐                       │
│  │ 귀 감지, 수염 감지, 눈 감지 │                       │
│  └─────────────────────────────┘                       │
│       ↓                                                 │
│  Layer 2: 특징 조합                                     │
│  ┌─────────────────────────────┐                       │
│  │ 고양이 얼굴 패턴 인식       │                       │
│  └─────────────────────────────┘                       │
│       ↓                                                 │
│  Layer 3: 분류                                          │
│  ┌─────────────────────────────┐                       │
│  │ 고양이: 0.85                │                       │
│  │ 강아지: 0.10                │                       │
│  │ 새:     0.05                │                       │
│  └─────────────────────────────┘                       │
│       ↓                                                 │
│  예측: "고양이" (85% 확신)                              │
│                                                         │
└─────────────────────────────────────────────────────────┘

```




### Backward Pass 상세
```
┌─────────────────────────────────────────────────────────┐
│                    Backward Pass                        │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  정답: "고양이"                                         │
│  예측: "고양이" 85%, "강아지" 10%, "새" 5%              │
│  Loss = 0.163 (낮을수록 좋음)                           │
│       ↑                                                 │
│  "어떻게 하면 Loss를 줄일 수 있을까?"                   │
│       ↑                                                 │
│  Layer 3 분석:                                          │
│  ┌─────────────────────────────┐                       │
│  │ "고양이 점수를 더 높여야 함" │                       │
│  │ W₃를 조금 증가시키자         │                       │
│  │ ∇W₃ = +0.02                 │                       │
│  └─────────────────────────────┘                       │
│       ↑                                                 │
│  Layer 2 분석:                                          │
│  ┌─────────────────────────────┐                       │
│  │ "얼굴 패턴 인식을 강화"      │                       │
│  │ W₂를 조정                    │                       │
│  │ ∇W₂ = +0.015                │                       │
│  └─────────────────────────────┘                       │
│       ↑                                                 │
│  Layer 1 분석:                                          │
│  ┌─────────────────────────────┐                       │
│  │ "귀 감지 강화, 수염 감지 강화"│                      │
│  │ W₁를 조정                    │                       │
│  │ ∇W₁ = +0.008                │                       │
│  └─────────────────────────────┘                       │
│       ↑                                                 │
│  모든 gradient 계산 완료!                               │
│                                                         │
└─────────────────────────────────────────────────────────┘

```
